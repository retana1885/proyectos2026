# Portafolio de Proyectos — J. Pablo Retana L.

Portafolio personal de proyectos de desarrollo. Muestra estatus de avance, stack técnico, capturas de pantalla e hitos por proyecto. Diseño oscuro, dinámico desde JSON.

🔗 **[Ver en vivo](https://retana1885.github.io/proyectos2026/)**

---

## Stack

HTML · CSS · JavaScript vanilla — sin frameworks, sin dependencias.  
Los datos se leen desde `data/projects.json` vía `fetch()`.

## Correr localmente

```bash
python -m http.server 8000
# Abrir http://localhost:8000
```

## Proyectos incluidos

14 proyectos de desarrollo: aplicaciones web, herramientas de escritorio, bots, scrapers y dashboards construidos con Python, Flask, FastAPI, JavaScript y más.

## Seguridad y privacidad

Este repositorio es público. Se tomaron las siguientes medidas antes de publicarlo:

- **Sin credenciales ni contraseñas** — ningún archivo contiene passwords, API keys, tokens ni secrets.
- **Sin IPs ni URLs internas** — las direcciones de servidores, bases de datos y redes privadas (Tailscale, VPN) están excluidas del repo via `.gitignore`.
- **Sin datos de clientes** — las capturas de pantalla fueron revisadas antes de subirse; no contienen nombres reales, montos, folios ni información operativa sensible.
- **Sin configuraciones de infraestructura** — archivos `.env`, `config.json` y scripts de despliegue son locales y no forman parte del repositorio.
- **Datos del portafolio** — `data/projects.json` contiene únicamente descripciones técnicas de los proyectos: stack, avance, rol y fechas. Nada de lógica de negocio ni datos de producción.
